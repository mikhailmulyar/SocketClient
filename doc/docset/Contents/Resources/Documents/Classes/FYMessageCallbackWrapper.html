FYMessageCallbackWrapper Class Reference

Inherits from NSObject
Declared in FYClient.m

Overview

Blocks needs to be copied, when stored. Because you can use one single block as callback for multiple subscripted channels and we don’t want to copy the block for each channel, use a wrapper object, which copy the block once and which will be released first, if all references, which are held each by a channel, were released by unsubscripting all these channels.
