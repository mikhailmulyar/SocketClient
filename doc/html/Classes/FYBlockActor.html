Class Methods

chain:once:restore:

Chain an actor once with a block an after the block was executed one single time, remove the chain from its place by executing a restore block where the original actor will be given as argument.

+ (instancetype)chain:(id<FYActor>)actor once:(FYActorBlock)actorBlock restore:(void ( ^ ) ( id<FYActor> ))restoreBlock

Parameters

actor

Actor to be chained once. This will be executed before actorBlock and given as first argument to restoreBlock.

actorBlock

Block to execute once.

restoreBlock

Restore action to replace return value back by actor.

Discussion

Chain an actor once with a block an after the block was executed one single time, remove the chain from its place by executing a restore block where the original actor will be given as argument.

Return value must replace parameter actor to take effect.

 self.connectActor = [FYBlockActor chain:self.connectActor
                                    once:^(FYClient *client, FYMessage *message){
                                        NSLog(@"This is done once.");
                                    } restore:^(idFYActor actor){
                                        self.connectActor = actor;
                                    }]
